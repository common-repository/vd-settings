== Description ==

Viaduct Essential Options is a small plugin which helps admins to easily enable some of the options without manually adding them into functions.php and instead providing a regular settings page in the WordPress admin panel.

1. Options provided by the plugin

== Installation ==

1. Upload the 'vd-settings' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
